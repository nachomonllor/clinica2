## Comandos de heroku

1. Instalar el toolkit heroku
   https://devcenter.heroku.com/articles/heroku-cli

2. $ heroku login

3. Crear una app 
   $ heroku create

4. Renombrar
   $ heroku apps:rename --app [nombreviejo] [nombrenuevo] 

5. git remote -v
   ver si esta el repo de heroku

6. git push heroku master
7. heroku logs --tail
